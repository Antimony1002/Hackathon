你是一个全栈开发师，擅长使用HTML5、CSS3、JavaScript构建网页，请用 HTML + Tailwind CSS 复刻一个名为 「交我通」 的学习辅助网站首页，要求如下：
# 1. 视觉还原：
- 高度还原桌面版 UI 设计风格，整体风格简洁、专注、高效，面向大学生用户。
- **主题色**：#3B82F6（Tailwind 的 blue-500），辅以中性灰（bg-gray-50, text-gray-700）。
- **顶部导航栏**：
    - 左侧 Logo：“交我通”
    - 中间核心导航链接：首页、我的资料、模拟测试、复习计划
    - 导航栏固定在顶部（fixed top-0），背景带轻微毛玻璃效果（可用 backdrop-blur-sm 模拟）
- **主要内容区布局**：
    - 采用 两栏布局：左侧为主内容区（占 2/3），右侧为智能助手面板（占 1/3）
    - 主内容区包含以下功能区块：
    - **资料上传区（卡片式）**：支持上传“考试范围文档”、“课本 PDF”、“教师 PPT”、“历年真题 ZIP”、“可用复习时长”、“目标分数”等
    - 智能生成按钮：“一键生成复习方案”
- 右侧助手面板：
  显示“推荐每日任务”、“进度条（0%）”
- 设计细节：
  卡片使用 rounded-xl 圆角 + shadow-md 阴影
  按钮使用 px-6 py-3 rounded-lg font-medium，主按钮为蓝色实心，次要按钮为白色描边
  所有交互元素需有悬停反馈（如按钮 hover 变深蓝，卡片 hover 微微上浮）

# 3. 技术实现：
    - 单个 HTML 文件完成全部结构与样式。
- **桌面端宽度**  ：容器最大宽度 max-w-7xl（约 1440px），居中布局。
- **桌面端**：两栏布局
- **图标库**：通过 CDN 引入 Font Awesome 5，用于上传图标（fas fa-file-pdf）、时钟（fas fa-clock）、进度（fas fa-tasks）等。
# 4. 关键功能/交互（静态页面模拟）：
- 固定顶部导航栏（position: fixed）
- **悬停交互**：
- 资料卡片 hover 时：transform: translateY(-2px) + shadow-lg
- 主按钮 hover 时：背景色变为 #2563eb（blue-600）
- 动态展示模拟（无需真实逻辑，仅视觉示意）：

- **表单元素**：
    - 包含一个隐藏的文件上传 input（视觉上用“+ 添加资料”卡片代替）
